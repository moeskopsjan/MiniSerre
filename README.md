# MiniSerre

In dit project delen we hoe je met een Arduino, temperatuursensor (DS18b20) en een mini servo een miniserre kan besturen.  Deze miniserre kan men voor een raam in het zonlicht plaatsen.  Bij een bepaalde temperatuur zal de miniserre bovenaan geopend worden met de mini servo.

Als uitbreiding kan je ook een LCD 1602 met I2C module aansluiten, zodat je de temperatuur en de hoek van de servo kan controlleren.
